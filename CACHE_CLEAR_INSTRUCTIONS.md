# Инструкция по очистке кэша

## Проблема
Браузер показывает старую дефолтную страницу Vite вместо нашего приложения.

## Решение - очистите кэш браузера

### Вариант 1: Жесткая перезагрузка (самый простой)

1. Откройте страницу http://localhost:5173
2. Нажмите **Ctrl + Shift + R** (или **Ctrl + F5**)
3. Или **Shift + F5**

### Вариант 2: Очистка кэша в Chrome/Edge

1. Нажмите **F12** (откроется DevTools)
2. Нажмите **Ctrl + Shift + Delete**
3. Выберите "Изображения и другие файлы, сохраненные в кеше"
4. Нажмите "Удалить данные"
5. Обновите страницу **F5**

### Вариант 3: Режим инкогнито

1. Нажмите **Ctrl + Shift + N** (Chrome/Edge)
2. Откройте http://localhost:5173
3. Проверьте, работает ли там

### Вариант 4: Полная очистка (если ничего не помогло)

#### Для Chrome/Edge:

1. Настройки → Конфиденциальность и безопасность
2. Удалить данные о просмотре
3. Выберите:
   - Файлы cookie
   - Кэш
4. Период: За все время
5. Удалить данные

#### Для Firefox:

1. Настройки → Приватность и защита
2. Удалить данные
3. Выберите все
4. Удалить

## Что должно быть видно после очистки:

✅ Голубая шапка сверху
✅ Белая карточка в центре
✅ Страница входа или dashboard

❌ НЕ должно быть:
- Логотипов Vite и TypeScript
- Текста "count is 0"
- Текста "Click on the Vite and TypeScript logos"
