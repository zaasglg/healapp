Supabase Studio не может подключиться к БД, потому что:
- Пользователь `supabase_admin` существует, но пароль неправильный
- Meta сервис не может подключиться к БД
- Ошибка: `password authentication failed for user "supabase_admin"`

## Решение

Пароль для `supabase_admin` обновлен через ALTER USER:
- Пароль установлен: `Dn2907200!`
- Meta сервис перезапущен

Если вы сбросите пароль через интерфейс Studio, нужно будет также:
1. Обновить его в `.env` файле (переменная `SUPABASE_DB_PASSWORD`)
2. Обновить пароль для `supabase_admin` через SQL
3. Перезапустить контейнеры
