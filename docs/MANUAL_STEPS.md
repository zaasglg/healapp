# Что нужно сделать вручную

## 1. Использовать IP вместо домена

Домен `supabase.healapp.ru` не резолвится с вашего компьютера. Используйте IP напрямую:

**Откройте в браузере:** `http://176.124.217.224/`

## 2. Очистить кэш браузера

После открытия по IP:

1. Нажмите `F12` (откроется DevTools)
2. Правой кнопкой на кнопку обновления страницы
3. Выберите **"Очистить кэш и жесткая перезагрузка"** (Empty Cache and Hard Reload)
4. Или используйте `Ctrl+Shift+R` (Windows) / `Cmd+Shift+R` (Mac)

## 3. Войти в Studio

После очистки кэша:

1. Появится окно запроса логина/пароля
2. Введите:
   - **Логин**: `admin`
   - **Пароль**: `HealApp2024SecurePass!`
3. После входа должны отображаться все таблицы

## 4. Если таблицы все еще не отображаются

Попробуйте режим инкогнито:

1. Откройте браузер в режиме инкогнито (`Ctrl+Shift+N`)
2. Перейдите на `http://176.124.217.224/`
3. Введите логин/пароль: `admin` / `HealApp2024SecurePass!`

## Важно

- **Используйте IP**: `http://176.124.217.224/` (не домен)
- **Очистите кэш** перед повторной попыткой
- **Self-hosted Supabase** работает с одним проектом - все данные БД доступны после входа
