# Система обучения пользователей платформы

## Общая структура

### Расположение
- **Местоположение**: Страница профиля (`/profile`)
- **Элемент интерфейса**: Плашка "Как пользоваться платформой"
- **Формат**: Вкладки с текстовым описанием и скриншотами

### Структура обучения
Каждая вкладка содержит:
1. **Описание функции** - для чего нужна эта функция, какую проблему решает
2. **Текстовое руководство** - пошаговая инструкция по использованию
3. **Скриншоты** - визуальные примеры интерфейса

---

## Обучение для ПАНСИОНАТОВ

### Вкладка 1: Создание карточки подопечного

**Описание функции:**
Карточка подопечного - это базовая информация о человеке, за которым вы ухаживаете. Она содержит личные данные, диагнозы, особенности ухода и пожелания по услугам. Карточка необходима для создания дневника наблюдений и позволяет систематизировать информацию о подопечном.

**Как создать карточку:**
1. Перейдите в раздел "Профиль" → "Карточки подопечных"
2. Нажмите кнопку "+ Добавить карточку" внизу экрана
3. Заполните обязательные поля:
   - ФИО подопечного
   - Дата рождения
   - Пол
   - Адрес проживания
4. Укажите дополнительные данные:
   - Подъезд и квартиру (если применимо)
   - Диагнозы (выберите из списка или добавьте свои)
   - Подвижность (ходит, сидит, лежит)
   - Наличие домашних животных
   - Необходимые услуги
   - Пожелания по услугам
5. Нажмите "Готово" для сохранения карточки

**Скриншоты:**
- Главный экран "Карточки подопечных"
- Форма создания карточки (обязательные поля)
- Форма создания карточки (дополнительные поля)
- Сохраненная карточка в списке

### Вкладка 2: Создание дневника подопечного

**Описание функции:**
Дневник подопечного - это инструмент для ежедневного мониторинга состояния здоровья и ухода. В дневнике фиксируются показатели здоровья (давление, температура, пульс), выполненные процедуры, прием лекарств и другие важные данные. Дневник помогает отслеживать динамику состояния подопечного и обеспечивает преемственность ухода между сменами.

**Как создать дневник:**
1. Перейдите в раздел "Мои дневники" на главной странице
2. Нажмите кнопку "+ Создать новый дневник" внизу экрана
3. Выберите карточку подопечного из списка (для которой еще не создан дневник)
4. Настройте показатели дневника:
   - Выберите категории показателей (Уход, Физические параметры, Выделения, Симптомы)
   - Выберите конкретные показатели в каждой категории
   - Закрепите важные показатели для быстрого доступа
   - Добавьте свои пользовательские показатели при необходимости
5. Нажмите "Создать дневник" для завершения

**Скриншоты:**
- Экран выбора карточки для дневника
- Экран настройки показателей (категории)
- Экран выбора конкретных показателей
- Экран закрепления показателей
- Созданный дневник в списке

### Вкладка 3: Добавление сотрудника в организацию

**Описание функции:**
Добавление сотрудников позволяет расширить команду ухода и распределить обязанности. Каждый сотрудник получает доступ к дневникам подопечных в зависимости от своей роли. Это обеспечивает командную работу и контроль качества ухода.

**Как добавить сотрудника:**
1. Перейдите в раздел "Профиль" → "Сотрудники"
2. В разделе "Пригласить специалиста" выберите роль сотрудника:
   - Администратор
   - Руководитель
   - Сиделка
   - Врач
3. Нажмите "Создать пригласительную ссылку"
4. Ссылка автоматически скопируется в буфер обмена
5. Отправьте ссылку сотруднику через мессенджер или email
6. После регистрации сотрудник появится в списке "Сотрудники"

**Скриншоты:**
- Экран "Сотрудники" с выбором роли
- Созданная пригласительная ссылка
- Список активных приглашений
- Список зарегистрированных сотрудников

---

### Вкладка 4: Управление доступом сотрудников к дневнику

**Описание функции:**
Управление доступом позволяет контролировать, какие сотрудники могут просматривать и заполнять конкретный дневник. Для пансионатов все сотрудники организации автоматически имеют доступ ко всем дневникам организации. Вы можете просмотреть список сотрудников с доступом, но не можете ограничивать доступ (это особенность пансионатов).

**Как управлять доступом:**
1. Откройте нужный дневник подопечного
2. Нажмите на иконку настроек (шестеренка) в правом верхнем углу
3. Перейдите во вкладку "Доступ"
4. В разделе "Сотрудники организации" вы увидите список всех сотрудников с доступом к дневнику
5. Для пансионатов все сотрудники организации автоматически имеют доступ

**Скриншоты:**
- Экран настроек дневника
- Вкладка "Доступ" с разделом сотрудников
- Список сотрудников с доступом

---

### Вкладка 5: Предоставление доступа клиенту к дневнику

**Описание функции:**
Предоставление доступа клиенту позволяет самому подопечному или его родственникам просматривать дневник наблюдений. Это обеспечивает прозрачность ухода и позволяет клиенту быть в курсе состояния здоровья. Клиент может просматривать все записи, но не может редактировать их.

**Как дать доступ клиенту:**
1. Откройте нужный дневник подопечного
2. Нажмите на иконку настроек (шестеренка) в правом верхнем углу
3. Перейдите во вкладку "Доступ"
4. В разделе "Доступ для клиента" нажмите "Создать пригласительную ссылку"
5. Ссылка автоматически скопируется в буфер обмена
6. Отправьте ссылку клиенту через мессенджер или email
7. После регистрации клиент получит доступ к дневнику

**Скриншоты:**
- Экран настроек дневника
- Вкладка "Доступ" с разделом для клиента
- Создание пригласительной ссылки для клиента
- Статус доступа клиента

### Вкладка 6: Использование дневника подопечного (закрепленные параметры)

**Описание функции:**
Закрепленные параметры - это важные показатели, которые отображаются в верхней части дневника для быстрого доступа. Они позволяют быстро вносить ежедневные измерения (давление, температура, пульс и т.д.) без необходимости искать их в полном списке показателей.

**Как использовать закрепленные параметры:**
1. Откройте дневник подопечного
2. В верхней части экрана вы увидите панели с закрепленными параметрами
3. Для внесения измерения:
   - Нажмите на панель нужного параметра
   - Введите значение измерения
   - При необходимости выберите время напоминания (если нужно измерять несколько раз в день)
   - Нажмите кнопку "+" для добавления времени измерения
   - Нажмите "Сохранить" для записи значения
4. Все внесенные значения отображаются в истории измерений

**Скриншоты:**
- Дневник с закрепленными параметрами вверху
- Панель параметра с полем ввода
- Добавление времени измерения
- История измерений параметра

---

### Вкладка 7: Использование дневника подопечного (все параметры)

**Описание функции:**
Все параметры дневника включают полный список показателей, разделенных по категориям: Уход, Физические параметры, Выделения, Симптомы. Это позволяет фиксировать любые данные о состоянии подопечного и выполненных процедурах.

**Как использовать все параметры:**
1. Откройте дневник подопечного
2. Прокрутите вниз до раздела "Все параметры"
3. Выберите нужную категорию (Уход, Физические параметры, Выделения, Симптомы)
4. Нажмите на нужный параметр
5. Внесите значение:
   - Для числовых параметров (давление, температура) введите значение
   - Для булевых параметров (да/нет) выберите соответствующую кнопку
   - Для текстовых параметров введите описание
6. Нажмите "Сохранить" для записи

**Скриншоты:**
- Раздел "Все параметры" в дневнике
- Категории параметров
- Форма ввода значения параметра
- Сохраненная запись в истории

### Вкладка 8: Редактирование дневников и карточек

**Описание функции:**
Редактирование позволяет обновлять информацию о подопечном и настройки дневника. Это необходимо при изменении состояния здоровья, адреса, диагнозов или при необходимости добавить новые показатели в дневник.

**Как редактировать карточку:**
1. Перейдите в раздел "Профиль" → "Карточки подопечных"
2. Нажмите на нужную карточку
3. Внесите необходимые изменения
4. Нажмите "Сохранить" для применения изменений

**Как редактировать дневник:**
1. Откройте нужный дневник подопечного
2. Нажмите на иконку настроек (шестеренка) в правом верхнем углу
3. Во вкладке "Карточка" можно изменить данные карточки подопечного
4. Во вкладке "Показатели" можно:
   - Добавить новые показатели
   - Удалить ненужные показатели
   - Изменить закрепленные показатели
5. Сохраните изменения

**Скриншоты:**
- Редактирование карточки
- Настройки дневника (вкладка "Карточка")
- Настройки дневника (вкладка "Показатели")
- Добавление/удаление показателей

### Вкладка 9: Роли сотрудников и их права

**Описание функции:**
Понимание ролей сотрудников помогает правильно распределить обязанности в команде и обеспечить эффективную работу. Каждая роль имеет определенный набор прав доступа к функциям платформы.

**Роли и права:**

**Администратор:**
- Создание дневников и карточек подопечных
- Изменение дневников и карточек
- Заполнение дневников
- Просмотр всех дневников организации

**Руководитель:**
- Создание дневников и карточек подопечных
- Редактирование дневников и карточек
- Раздача доступа для клиентов
- Управление доступом для сотрудников к дневнику
- Заполнение дневников
- Просмотр всех дневников организации

**Сиделка:**
- Просмотр назначенных дневников
- Заполнение назначенных дневников
- НЕ может создавать дневники и карточки
- НЕ может редактировать дневники и карточки
- НЕ может управлять доступом

**Врач:**
- Просмотр назначенных дневников
- Заполнение назначенных дневников
- НЕ может создавать дневники и карточки
- НЕ может редактировать дневники и карточки
- НЕ может управлять доступом

**Скриншоты:**
- Экран создания приглашения с выбором роли
- Описание прав для каждой роли (из интерфейса)

### Обучение для АДМИНИСТРАТОРОВ

#### Вкладка 1: Создание дневников и карточек

**Описание функции:**
Как администратор, вы можете создавать карточки подопечных и дневники наблюдений. Это позволяет вам самостоятельно добавлять новых подопечных в систему и настраивать для них дневники наблюдений.

**Как создать карточку:**
1. Перейдите в раздел "Профиль" → "Карточки подопечных"
2. Нажмите кнопку "+ Добавить карточку"
3. Заполните все необходимые поля
4. Нажмите "Готово" для сохранения

**Как создать дневник:**
1. Перейдите в раздел "Мои дневники"
2. Нажмите "+ Создать новый дневник"
3. Выберите карточку подопечного
4. Настройте показатели дневника
5. Нажмите "Создать дневник"

**Скриншоты:**
- Создание карточки (форма)
- Создание дневника (выбор карточки)

**Как редактировать:**
1. Откройте карточку или дневник
2. Нажмите на иконку редактирования или перейдите в настройки
3. Внесите изменения
4. Сохраните изменения

**Скриншоты:**
- Редактирование карточки
- Настройки дневника

#### Вкладка 3: Заполнение дневника

**Описание функции:**
Заполнение дневника - это ваша основная задача. Вы фиксируете все важные данные о состоянии подопечного: показатели здоровья, выполненные процедуры, прием лекарств и другие наблюдения.

**Как заполнять дневник:**
1. Откройте дневник подопечного из списка "Мои дневники"
2. Используйте закрепленные параметры для быстрого ввода частых измерений
3. Используйте раздел "Все параметры" для ввода других данных
4. Регулярно вносите данные для обеспечения непрерывного мониторинга

**Скриншоты:**
- Список дневников

#### Вкладка 1: Просмотр назначенных дневников

**Описание функции:**
Вы видите только те дневники, к которым вас назначил руководитель организации. Это позволяет вам сосредоточиться на работе с конкретными подопечными.

**Как просматривать дневники:**
1. Перейдите в раздел "Мои дневники" на главной странице
2. Вы увидите список дневников, к которым у вас есть доступ
3. Нажмите на дневник для открытия

**Скриншоты:**
- Список доступных дневников
- Открытый дневник

#### Вкладка 2: Заполнение дневника

**Описание функции:**
Ваша основная задача - заполнять дневник подопечного. Вы фиксируете показатели здоровья, выполненные процедуры и другие важные данные о состоянии подопечного.

**Как заполнять дневник:**
1. Откройте дневник подопечного
2. Используйте закрепленные параметры для быстрого ввода частых измерений:
   - Нажмите на панель параметра
   - Введите значение
   - При необходимости добавьте время измерения (кнопка "+")
   - Нажмите "Сохранить"
3. Используйте раздел "Все параметры" для ввода других данных:
   - Выберите категорию параметров
   - Выберите конкретный параметр
   - Введите значение
   - Нажмите "Сохранить"

**Скриншоты:**
- Дневник с закрепленными параметрами
- Ввод значения в закрепленный параметр
- Раздел "Все параметры"
- Ввод значения в параметр из категории
- История записей

#### Вкладка 3: Ограничения вашей роли

**Описание функции:**
Как врач или сиделка, вы имеете ограниченный набор прав. Это сделано для обеспечения контроля качества и безопасности данных.

**Что вы МОЖЕТЕ делать:**
- Просматривать назначенные дневники
- Заполнять назначенные дневники
- Просматривать историю записей

**Что вы НЕ МОЖЕТЕ делать:**
- Создавать новые дневники
- Создавать карточки подопечных
- Редактировать дневники и карточки
- Управлять доступом к дневникам
- Предоставлять доступ клиентам

**Скриншоты:**
- Пример недоступной функции (кнопка создания дневника скрыта)
- Пример недоступной функции (настройки дневника недоступны)

## Обучение для КЛИЕНТОВ

### Вкладка 1: Регистрация по приглашению

**Описание функции:**
Если вас пригласила организация или частная сиделка, после регистрации вы автоматически получаете доступ к дневнику подопечного. Это позволяет вам быть в курсе состояния здоровья и ухода за вашим близким человеком.

**Что происходит после регистрации:**
1. После перехода по пригласительной ссылке и регистрации
2. Вы автоматически получаете доступ к дневнику подопечного
3. Организация или сиделка, которая вас пригласила, также имеет доступ к дневнику
4. Вы можете просматривать все записи в дневнике, но не можете их редактировать

**Скриншоты:**
- Экран регистрации по приглашению
- Список дневников после регистрации
- Открытый дневник с записями

---

### Вкладка 2: Создание карточки подопечного

**Описание функции:**
Если вы хотите самостоятельно вести дневник наблюдений за собой или за близким человеком, вы можете создать карточку подопечного. Карточка содержит основную информацию о подопечном и необходима для создания дневника.

**Как создать карточку:**
1. Перейдите в раздел "Профиль" → "Карточки подопечных"
2. Нажмите кнопку "+ Добавить карточку" внизу экрана
3. Заполните обязательные поля:
   - ФИО подопечного
   - Дата рождения
   - Пол
   - Адрес проживания
4. Укажите дополнительные данные (диагнозы, подвижность, услуги и т.д.)
5. Нажмите "Готово" для сохранения

**Скриншоты:**
- Экран "Карточки подопечных"
- Форма создания карточки
- Сохраненная карточка

---

### Вкладка 3: Создание дневника

**Описание функции:**
Дневник позволяет вам самостоятельно вести наблюдения за состоянием здоровья. Вы можете фиксировать показатели здоровья, прием лекарств, симптомы и другие важные данные.

**Как создать дневник:**
1. Перейдите в раздел "Мои дневники" на главной странице
2. Нажмите кнопку "+ Создать новый дневник"
3. Выберите карточку подопечного (для которой еще не создан дневник)
4. Настройте показатели дневника:
   - Выберите категории показателей
   - Выберите конкретные показатели
   - Закрепите важные показатели
5. Нажмите "Создать дневник"

**Скриншоты:**
- Экран создания дневника (выбор карточки)
- Настройка показателей дневника
- Созданный дневник

---

### Вкладка 4: Управление доступом к дневнику

**Описание функции:**
Управление доступом позволяет вам контролировать, кто может просматривать дневник. Вы можете предоставить доступ организациям, частным сиделкам или родственникам. Это обеспечивает прозрачность и позволяет другим людям быть в курсе состояния здоровья.

**Как управлять доступом:**
1. Откройте нужный дневник
2. Нажмите на иконку настроек (шестеренка) в правом верхнем углу
3. Перейдите во вкладку "Доступ"
4. В разделе "Доступ для организации/сиделки":
   - Нажмите "Создать пригласительную ссылку"
   - Отправьте ссылку организации или сиделке
5. В разделе "Доступ для клиента":
   - Нажмите "Создать пригласительную ссылку"
   - Отправьте ссылку родственнику или другому лицу

**Скриншоты:**
- Настройки дневника (вкладка "Доступ")
- Создание пригласительной ссылки
- Список пользователей с доступом

---

### Вкладка 5: Отзыв доступа

**Описание функции:**
Вы можете отозвать доступ к дневнику у любого пользователя, которому вы его предоставили. Это позволяет вам контролировать, кто имеет доступ к информации о состоянии здоровья.

**Как отозвать доступ:**
1. Откройте нужный дневник
2. Перейдите в настройки (иконка шестеренки) → вкладка "Доступ"
3. Найдите пользователя в списке тех, кому предоставлен доступ
4. Нажмите на имя пользователя
5. Выберите "Отозвать доступ"
6. Подтвердите действие

**Скриншоты:**
- Список пользователей с доступом
- Отзыв доступа (кнопка/меню)
- Подтверждение отзыва доступа

---

### Вкладка 6: Предоставление доступа другим (родственники, организации, сиделки)

**Описание функции:**
Вы можете предоставить доступ к дневнику нескольким людям одновременно. Это полезно, если за подопечным ухаживают несколько организаций, сиделок или если вы хотите, чтобы родственники были в курсе состояния здоровья.

**Как дать доступ:**
1. Откройте нужный дневник
2. Перейдите в настройки (иконка шестеренки) → вкладка "Доступ"
3. Создайте пригласительную ссылку:
   - Для организации/сиделки: используйте раздел "Доступ для организации/сиделки"
   - Для родственника: используйте раздел "Доступ для клиента"
4. Отправьте ссылку нужному человеку или организации
5. После регистрации они получат доступ к дневнику

**Важно:**
- Вы можете создать несколько пригласительных ссылок
- Каждая ссылка может быть использована только один раз
- Вы всегда можете отозвать доступ у любого пользователя

**Скриншоты:**
- Настройки дневника (вкладка "Доступ")
- Создание пригласительной ссылки для организации
- Создание пригласительной ссылки для клиента
- Список всех пользователей с доступом

---

### Вкладка 7: Использование дневника (закрепленные параметры)

**Описание функции:**
Закрепленные параметры отображаются в верхней части дневника для быстрого доступа. Они позволяют быстро вносить ежедневные измерения без необходимости искать их в полном списке.

**Как использовать:**
1. Откройте дневник
2. В верхней части экрана найдите панели с закрепленными параметрами
3. Нажмите на панель нужного параметра
4. Введите значение измерения
5. При необходимости добавьте время напоминания (кнопка "+")
6. Нажмите "Сохранить"

**Скриншоты:**
- Дневник с закрепленными параметрами
- Ввод значения в закрепленный параметр
- История измерений

---

### Вкладка 8: Использование дневника (все параметры)

**Описание функции:**
Все параметры включают полный список показателей, разделенных по категориям. Это позволяет фиксировать любые данные о состоянии здоровья.

**Как использовать:**
1. Откройте дневник
2. Прокрутите вниз до раздела "Все параметры"
3. Выберите категорию (Уход, Физические параметры, Выделения, Симптомы)
4. Нажмите на нужный параметр
5. Введите значение
6. Нажмите "Сохранить"

**Скриншоты:**
- Раздел "Все параметры"
- Категории параметров
- Ввод значения параметра
- История записей

## Технические требования к реализации

### Компоненты интерфейса

1. **Плашка "Как пользоваться платформой"**
   - Расположение: Страница профиля (`/profile`)
   - Стиль: Карточка с градиентным фоном или выделенным цветом
   - Иконка: Книга/учебник или иконка "?"

2. **Вкладки обучения**
   - Формат: Табы (вкладки) с названиями разделов
   - Навигация: Горизонтальная прокрутка или сетка вкладок
   - Активная вкладка: Выделена цветом

3. **Контент вкладки**
   - Заголовок раздела
   - Описание функции (блок с серым фоном или иконкой информации)
   - Текстовое руководство (нумерованный список или маркированный)
   - Скриншоты (изображения с подписями)

### Структура данных

```typescript
interface OnboardingSection {
  id: string
  title: string
  description: string // Описание функции
  steps: Array<{
    number: number
    text: string
  }>
  screenshots: Array<{
    url: string
    alt: string
    caption?: string
  }>
}

interface OnboardingContent {
  role: 'pension' | 'patronage_agency' | 'org_employee' | 'client' | 'caregiver'
  employeeRole?: 'admin' | 'manager' | 'caregiver' | 'doctor' // Для сотрудников
  sections: OnboardingSection[]
}
```

### Маршрутизация

- Основной маршрут: `/profile/onboarding`
- Параметры: `?role=pension&section=1` (для глубоких ссылок)

### Локализация

Все тексты должны быть на русском языке. В будущем можно добавить поддержку других языков.

## Примечания для разработчиков

1. **Скриншоты**: Все скриншоты должны быть актуальными и соответствовать текущему интерфейсу приложения
2. **Адаптивность**: Обучение должно корректно отображаться на мобильных устройствах
3. **Прогресс**: Можно добавить индикатор прогресса прохождения обучения
4. **Повторный просмотр**: Пользователь должен иметь возможность вернуться к обучению в любое время
5. **Обновления**: При изменении функционала необходимо обновлять соответствующие разделы обучения
