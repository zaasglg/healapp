# ✅ Финальное решение проблемы с паролем

## Проблема

Ошибка: `invalid_grant: Invalid login credentials`

GoTrue не может проверить пароль, даже если он установлен в правильном формате.

## Корневая причина

GoTrue использует PostgreSQL `crypt()` функцию для проверки паролей. Хеши, сгенерированные Python bcrypt, могут не работать, даже если формат правильный (`$2a$10$`).

## Решение

Использован PostgreSQL `crypt()` с `gen_salt('bf'::text)` для генерации хеша, который GoTrue может проверить напрямую.
