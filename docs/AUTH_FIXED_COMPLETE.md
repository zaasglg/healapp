## Проблема

Миграция `20221208132122_backfill_email_last_sign_in_at.up.sql` падала с ошибкой:
```
ERROR: operator does not exist: uuid = text
```

Проблемная строка в миграции:
```sql
id = user_id::text  -- ОШИБКА: UUID не может сравниваться с TEXT
```

## Решение

1. ✅ Создан кастомный Dockerfile для Auth с исправленной миграцией
2. ✅ Собран новый образ `gotrue-fixed:v2.132.3`
3. ✅ Обновлен `docker-compose.yml` для использования исправленного образа
4. ✅ Контейнер Auth пересоздан с новым образом

### Изменения в миграции

```sql
-- Было:
id = user_id::text

-- Стало:
id::text = user_id::text
```
