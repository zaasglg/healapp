# ✅ Исправлен порт Auth в nginx

## Проблема

Auth возвращал 404, потому что nginx проксировал на неправильный порт.

## Решение

**Было:** `proxy_pass http://127.0.0.1:54323/auth/;` (неправильный порт)
**Стало:** `proxy_pass http://127.0.0.1:54326/auth/;` (правильный порт)

Auth слушает на порту **54326** (внешний порт Docker), который маппится на внутренний порт 9999.

## Статус

- ✅ Порт Auth исправлен в nginx
- ✅ Nginx перезагружен
- ✅ Auth должен отвечать на запросы
