# Как проверить коммит в GitHub через Git Bash

## Способ 1: Проверка через Git Bash (локально)

1. **Откройте Git Bash** (если установлен Git, он обычно в меню Пуск → Git → Git Bash)

2. **Перейдите в папку проекта:**
   ```bash
   cd "/c/Users/User/Desktop/Веб версия дневник"
   ```

3. **Проверьте статус репозитория:**
   ```bash
   git status
   ```
   - Если есть незакоммиченные изменения, они будут показаны красным
   - Если все закоммичено, увидите: `nothing to commit, working tree clean`

4. **Посмотрите последние коммиты:**
   ```bash
   git log --oneline -5
   ```
   - Покажет последние 5 коммитов с их хешами и сообщениями

5. **Проверьте, что изменения отправлены в GitHub:**
   ```bash
   git log origin/master --oneline -5
   ```
   - Покажет последние коммиты на удаленном репозитории

6. **Сравните локальную и удаленную ветки:**
   ```bash
   git log HEAD..origin/master --oneline
   ```
   - Если вывод пустой, значит локальная ветка синхронизирована с удаленной

## Способ 2: Проверка через веб-интерфейс GitHub

1. Откройте ваш репозиторий на GitHub: `https://github.com/nazardubnak/HealApp-Web`

2. Перейдите на вкладку **"Commits"** (или просто на главную страницу репозитория)

3. Проверьте последний коммит:
   - Должен быть виден коммит с сообщением: `"fix: изменен путь для Edge Functions на /var/functions"`
   - Дата и время коммита должны быть свежими

4. Откройте коммит и проверьте изменения:
   - Нажмите на хеш коммита
   - Убедитесь, что файл `Dockerfile.functions` изменен
   - Проверьте, что путь изменен с `/home/deno/functions` на `/var/functions`

## Способ 3: Проверка через команду (быстрая)

В Git Bash выполните:
```bash
cd "/c/Users/User/Desktop/Веб версия дневник"
git fetch origin
git log HEAD..origin/master --oneline
```

Если вывод пустой — все коммиты отправлены. Если есть коммиты — нужно выполнить `git push`.

## Что проверить в коммите:

✅ Файл `Dockerfile.functions` изменен  
✅ Путь изменен на `/var/functions`  
✅ Комментарий в коммите понятный  
✅ Дата коммита соответствует времени отправки
