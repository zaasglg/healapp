# Проверка данных и Storage

## Статус данных

Проверяю, восстановились ли данные в таблицах после восстановления из бэкапа.

## Статус Storage

Проверяю наличие bucket "avatars" для хранения аватарок пользователей.

## Что нужно сделать

### 1. Проверить данные в таблицах

Если данные не восстановились, нужно восстановить их из бэкапа заново.

### 2. Создать bucket "avatars"

Если bucket не существует, нужно создать его:

```sql
INSERT INTO storage.buckets (id, name, public) 
VALUES ('avatars', 'avatars', true) 
ON CONFLICT (id) DO NOTHING;
```

Bucket должен быть публичным (`public = true`), чтобы аватарки были доступны для просмотра.

## Политики Storage

Политики для bucket "avatars" должны быть в миграциях. Проверю, применены ли они.
