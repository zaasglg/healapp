# ✅ Пароль обновлен через PostgreSQL crypt()

## Проблема

GoTrue использует PostgreSQL `crypt()` функцию для проверки паролей, а не Python bcrypt. Поэтому хеши, сгенерированные Python, могут не работать.

## Решение

Использован PostgreSQL `crypt()` с `gen_salt('bf'::text)` для генерации хеша, который GoTrue может проверить.
