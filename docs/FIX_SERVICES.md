# Исправление сервисов REST и Auth

## Проблемы

1. **REST сервис**: ошибка пароля для подключения к БД
2. **Auth сервис**: отсутствует `API_EXTERNAL_URL`

## Решение

Нужно проверить и исправить конфигурацию в `docker-compose.yml`:
- `PGRST_DB_URI` для REST должен содержать правильный пароль
- `GOTRUE_API_EXTERNAL_URL` для Auth должен быть установлен
