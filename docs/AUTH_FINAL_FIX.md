# ✅ Исправлено проксирование Auth (финальная версия)

## Проблема

Auth слушает на `/v1/`, а не на `/auth/v1/`, поэтому запросы `/auth/v1/token` возвращали 404.

## Решение

Добавлен правильный `rewrite` для преобразования пути:

```nginx
rewrite ^/auth/v1/(.*) /v1/$1 break;
proxy_pass http://127.0.0.1:54326;
```

Теперь:
- Запрос `/auth/v1/token` → проксируется на `http://127.0.0.1:54326/v1/token` ✅
- Запрос `/auth/v1/settings` → проксируется на `http://127.0.0.1:54326/v1/settings` ✅
