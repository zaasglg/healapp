## Почему данные не видны в Table Editor?

**RLS (Row Level Security) включен для всех таблиц!**

Даже роль `postgres` (суперюзер) может быть ограничена RLS политиками в Supabase Studio.

### Решения:

1. **Используйте SQL Editor с ролью postgres:**
   - В SQL Editor выберите роль "postgres" (внизу справа)
   - Выполните запросы - данные будут видны

2. **Войдите через приложение:**
   - Используйте тестовые данные для входа
   - После входа данные будут видны согласно RLS политикам

3. **Временно отключите RLS (только для диагностики):**
   ```sql
   ALTER TABLE public.organizations DISABLE ROW LEVEL SECURITY;
   -- После проверки включите обратно:
   ALTER TABLE public.organizations ENABLE ROW LEVEL SECURITY;
   ```
